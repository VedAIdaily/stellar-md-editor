Stellar MD Editor

A refreshingly simple, free editor for .txt and .md files in Google Drive™.

Stellar MD Editor opens plain-text and Markdown files stored in your Google Drive, lets you edit them in a clean, distraction-free window, and saves them straight back to the same file in your Drive. It is a plain-text editor: you write and edit the Markdown source itself, and there is no formatted preview; your file renders wherever Markdown is displayed.

  • Use it from Drive: right-click a file → Open with → Stellar MD Editor
  • It can access only the files you open with it, nothing else in your Drive
  • No servers and no tracking: your text moves only between your browser and Google Drive
  • Free forever, no account needed
  • On a phone: open this page in your browser and tap Open a file from Drive
